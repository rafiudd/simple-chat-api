## Installation
1. Clone this repository or download zip
```bash
https://github.com/rafiudd/simple-chat-api
```
2. Install package
```bash
npm install
```
3. Import this db.sql to your local machine
4. Then run the project with 
```bash
node index.js or nodemon
```
